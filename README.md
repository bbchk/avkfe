Компания АВАК - производитель профессионального оборудования для обслуживания, а также поставщик и реализатор  автомобильных аккумуляторов в городе Винница с 2001 года.

Сегодня мы являемся единственным производителем в Украине контрольно-испытательного оборудования для сферы производства свинцово-кислотных источников тока (аккумуляторов) в Виннице. Выпускаемая аппаратура обладает высокой надежностью и долговечностью в эксплуатации благодаря использованию современной элементной базы, как правило, импортного производства, либо отечественных аналогов лучших производителей.
